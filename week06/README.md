File Systems and Storage Performance


This repository contains coursework for the File Systems and Storage Performance module.


The journal documents all tasks, explanations, and analysis.

Screenshots showing terminal commands and outputs are provided as evidence in a separate screenshots folder.


Structure

    journal/ – Coursework journal (PDF)

    screenshots/ – Evidence screenshots for all tasks

    data/ – Performance data (CSV)


All work was completed on a Linux system using command-line tools.
